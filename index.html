About Haskish

Haskish is a free online functional programming playground designed specifically for AQA A Level Computer Science students and teachers.

It provides a simplified Haskell-style environment that covers all the functional programming concepts required by the AQA specification (4.12), including:

  • List operators (cons :, concatenation ++, indexing !!)
  • Pattern matching and recursion
  • Higher-order functions (map, filter, fold)
  • Function composition
  • Partial application (currying)

Additionally, Haskish supports tuples, operator sections, guards, and lambda/anonymous functions for those wanting to explore further.

No installation required - just open your browser and start coding!

Built-in Functions

List Functions

  • head list - First element of a list
  • tail list - All but the first element
  • length list - Length of a list
  • null list - Check if list is empty
  • reverse list - Reverse a list
  • take n list - Take first n elements
  • drop n list - Drop first n elements

Higher-Order Functions

  • map fn list - Apply function to each element
  • filter predicate list - Filter list by predicate
  • fold fn acc list - Reduce list with accumulator

Arithmetic Functions

  • mod a b - Modulo (remainder)
  • div a b - Integer division
  • min a b - Minimum of two values
  • max a b - Maximum of two values

Logic Functions

  • not bool - Boolean negation

Other

  • error message - Throw an error
  • compose g f - Function composition (also g . f)
  • True / False - Boolean values (capitalized)
  • otherwise - Always true (for guards)
